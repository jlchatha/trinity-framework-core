Automated Verification

The Trinity Framework includes an automated verification script that checks many of the items on this list. Run it with:

$ cd ~/trinity/trinity-framework-core/tools/setup
$ ./verify-installation.sh ~/trinity

The script will check the structure, files, and permissions for all agent workspaces and report any issues.

Verification Report

The verification script generates a detailed report. A successful verification should look like this:

## Trinity Framework 2.0 Verification Report
## Generated: [Current Date and Time]

### Framework Repository
✓ Repository present at /home/user/trinity/trinity-framework-core
✓ All required directories present
